# MLB Statcast Feature Dictionary

This note explains the useful fields from the 2025 postseason pitch-by-pitch Statcast file.

Source file:

```text
/Users/cashmcdearis/Downloads/Data_MLB_2025_StatcastPostseason_PitchByPitch_20251102a.csv
```

## Game identifiers

- `game_pk`: unique game id.
- `game_date`: game date.
- `home_team`: home team abbreviation.
- `away_team`: away team abbreviation.
- `inning`: inning number.
- `inning_topbot`: `Top` means away team batting, `Bot` means home team batting.

## Score fields

- `home_score`: home score before pitch.
- `away_score`: away score before pitch.
- `post_home_score`: home score after pitch.
- `post_away_score`: away score after pitch.
- `bat_score`: batting team score before pitch.
- `fld_score`: fielding team score before pitch.

Use:

- derive final score per game,
- identify actual winner,
- measure run differential.

## Pitcher / batter

- `player_name`: pitcher name in the export.
- `pitcher`: pitcher id.
- `batter`: batter id.
- `stand`: batter handedness.
- `p_throws`: pitcher handedness.

Use:

- future pitcher matchup modeling,
- handedness splits,
- starter/bullpen identification if paired with roster data.

## Pitch quality

- `pitch_type`: code for pitch type.
- `pitch_name`: readable pitch name.
- `release_speed`: velocity.
- `effective_speed`: speed adjusted for release extension.
- `release_spin_rate`: spin rate.
- `release_extension`: extension.
- `spin_axis`: pitch spin axis.
- `pfx_x`: horizontal movement.
- `pfx_z`: vertical movement.
- `plate_x`: horizontal plate location.
- `plate_z`: vertical plate location.

Use:

- pitcher stuff quality,
- pitch mix,
- velocity changes,
- fatigue/change detection.

## Batted-ball quality

- `launch_speed`: exit velocity.
- `launch_angle`: launch angle.
- `hit_distance_sc`: estimated hit distance.
- `bb_type`: batted ball type.
- `estimated_ba_using_speedangle`: xBA.
- `estimated_woba_using_speedangle`: xwOBA.
- `woba_value`: actual wOBA value.
- `babip_value`: BABIP value.
- `iso_value`: ISO value.

Use:

- team offensive quality,
- pitcher contact suppression,
- hard-hit rate,
- expected production instead of raw runs only.

## Plate discipline

- `description`: pitch result description.
- `type`: broad pitch result type.
- `balls`, `strikes`: count.

Useful derived features:

- whiff rate = swinging strikes / swings
- CSW rate = called strikes + whiffs / total pitches
- chase/swing proxies if zone logic is added later
